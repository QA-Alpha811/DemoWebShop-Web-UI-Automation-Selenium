# DemoWebShop E-Commerce Web UI Automation Project

This repository contains a comprehensive Web UI Test Automation Framework built for the DemoWebShop e-commerce platform. The project is designed to validate end-to-end user journeys and critical e-commerce functionalities using robust automation patterns.

## 🚀 Key Framework Features & Implemented Concepts
*   **Page Object Model (POM):** Designed a highly maintainable and scalable framework architecture by separating UI elements from test scripts.
*   **Data-Driven Testing:** Implemented strategies to separate test data from code logic, enabling multiple test scenarios with different datasets.
*   **Robust Exception Handling:** Managed dynamic elements, pop-ups, and synchronization issues using advanced explicit wait strategies.
*   **Reusable Utilities:** Created a centralized custom utilities class for common operations like screen capture, js-executor, and element interactions.

## 🛠️ Tech Stack & Tools
*   **Language:** Java
*   **Automation Tool:** Selenium WebDriver
*   **Design Pattern:** Page Object Model (POM)
*   **Build & Dependency Management:** Maven
*   **Test Runner:** JUnit / TestNG

## 📋 Core Test Scenarios Covered
1.  **Shopping Cart & Checkout Journey:** Verifying adding products to the cart, modifying quantities, updating the cart, and completing a mock checkout process with address details.
2.  **Product Catalog Navigation:** Testing categories (Books, Computers, Electronics), sorting mechanisms, and product display views.
3.  **User Account Management:** Validating registration, login states, and profile/address updates.

---
*Note: This project was developed as part of a hands-on QA Automation program to practice structural framework design (POM) on realistic web applications.*
